# 枫语 / Feng Yu

个人技术博客，专注于人工智能、强化学习、因果推断与大语言模型。

## 技术栈

- **框架**: [Astro](https://astro.build)
- **主题**: 基于 [Firefly](https://github.com/CuteLeaf/Firefly) 修改
- **托管**: GitHub Pages

## 本地开发

```bash
pnpm install
pnpm dev
```
