🧭 AI 学习路径地图

从计算机基础到科研创新，构建完整的 AI 能力体系。路径覆盖编程、机器学习、深度学习、强化学习、因果推断、工程部署等关键模块。

⏱️ 学习周期：12-18 个月 📈 目标：技术能力 + 业务理解 + 工程落地 🎯 用户画像：高校学生 | 转行者 | 研究人员

📍 阶段化成长路线

Stage 0 · 计算机/数学基础

  • Python 编程、数据结构与算法
  • 线性代数、概率论、统计学
  • Git、Linux、SQL 基础指令
Python Linear Algebra Probability

Stage 1 · 机器学习核心

  • 监督、无监督、半监督学习
  • 模型评估、特征工程、调优
  • Scikit-learn、XGBoost、LightGBM
ML Basics Modeling Evaluation

Stage 2 · 深度学习进阶

  • 神经网络、CNN、RNN、Transformer
  • PyTorch / TensorFlow 框架实战
  • 模型压缩、部署、服务化
PyTorch Transformer Deployment

Stage 3 · 强化学习/因果推断

  • Q-Learning、Policy Gradient、Actor-Critic
  • 因果图、DoWhy、YLearn、DUCG 理论
  • 行业案例：推荐、广告、运筹优化
Reinforcement Learning Causal Inference DUCG

Stage 4 · 工程落地与科研

  • MLOps、数据治理、可解释性
  • 论文阅读、实验设计、性能对比
  • 写作与分享：博客、开源贡献、演讲
MLOps Explainability Research

🛣️ 里程碑规划

📘

第 0-3 个月 · 基础阶段

完成 2 门编程课程 + 1 本数学教材；掌握 5-10 个基础算法；搭建学习记录（Notion/GitHub）。

🧠

第 4-8 个月 · 模型阶段

完成 3 个不同场景的建模项目；输出 2 篇技术博客；参与 Kaggle/天池比赛并获得排名。

🚀

第 9-12 个月 · 实践阶段

实现 1 个完整工程项目，包含数据管道 + 模型服务；提交开源 PR 或发布 Python 包；参加技术分享。

🌟

第 13 个月以后 · 拓展阶段

深入强化学习、因果推断、图神经网络等方向；参与科研合作，准备论文投稿或高级面试。
