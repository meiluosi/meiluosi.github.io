⚡ 在线 Notebook 实验室

借助 Thebe + Binder 集成，可直接在浏览器中执行 Jupyter Notebook 代码段，无需本地环境。 适用于教学演示、算法实战、交互式学习等场景。下方示例展示了数学计算、数据可视化与强化学习单步模拟。

  • 第一次点击 "运行" 会唤醒远端内核，大约需要 30-60 秒。
  • 如需自定义依赖，请在 _config.yml 中更新 thebe.binder.repo 与相关配置。
  • 支持多段代码顺序执行，变量状态将被保留。
📌 温馨提示： 如果页面长时间停留在 "Launching server"，请确认配置的 Binder 仓库包含 requirements.txt 或 environment.yml，并能成功在 MyBinder 上启动。

1️⃣ 数学推导验证

使用 SymPy 进行符号求导，并格式化输出结果。


import sympy as sp

x, y = sp.symbols('x y')
expression = sp.sin(x) * sp.exp(y)
derivative = sp.diff(expression, x)

print("原始表达式：")
sp.pprint(expression)
print("\n对 x 求导：")
sp.pprint(derivative)

2️⃣ 数据可视化示例

通过 pandas + plotly express 绘制交互式散点图。


import pandas as pd
import plotly.express as px

url = "https://raw.githubusercontent.com/mwaskom/seaborn-data/master/iris.csv"
df = pd.read_csv(url)
fig = px.scatter(df,
                 x="sepal_length",
                 y="petal_length",
                 color="species",
                 title="Iris 数据集 - 花瓣与花萼尺寸关系")
fig.show()

3️⃣ 强化学习策略单步评估

随机策略在 CartPole 环境中执行单步，并输出更新后的状态。


import gymnasium as gym
import numpy as np

env = gym.make("CartPole-v1")
state, _ = env.reset(seed=42)
action = env.action_space.sample()
next_state, reward, terminated, truncated, _ = env.step(action)

print(f"采取动作: {action}")
print(f"奖励: {reward:.2f}")
print(f"是否终止: {terminated}")
print("下一状态:")
print(np.array2string(next_state, formatter={'float_kind':lambda v: f"{v:7.3f}"}))
